# Traffic Simulation

A traffic simulation engine using C++ and raylib for my final year uiniversity project.

## Setup Instructions:

Ensure you have CMake and a C++ compiler installed.

Make a build directory using `mkdir build`.
Enter the directory: `cd build`.
Run `cmake ..` to configure the project.

Build by entering build directory and running `cmake --build .`.

Run the program by running `./TrafficSimulator`.
